Excel to XML

How to convert Joinery Scene Excel report to XML:

  • 1- The Excel file must be generated by the Microvellum report "01 - Excel Export"
  • 2- Save the report as Excel (.xls)
  • 3- Name it with the job name (the file name will be used to generate the Project name)
  • - - - Example: 01-Project Name (1234).xls
  • 4- Click on the "Browse..." button on this page
  • 5- Select the Excel from step 2
  • 6- Click on the download button on top of this page
  • 7- On Microvellum / Micromanager go to "Toolbox setup" -> "Import File"
  • 8- Click on the ellipsis (...) button on the File Name section
  • 9- Finally, click on the import button at the bottom
  • 10- In order to convert another file, click on the "Clear" button and repeat the process
